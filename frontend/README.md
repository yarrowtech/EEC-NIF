# Frontend (React + Vite)

## Setup

1. Create env file:
   - Copy `.env.example` to `.env`
2. Set backend URL:
   - `VITE_API_URL=http://localhost:5000`

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Build

```bash
npm run preview
```
